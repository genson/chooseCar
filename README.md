#Choose car
"Choose a car" is a small angular-application ( game ). The player is offered to guess the image of the machine based on its name.

* Demo: http://genson.github.io/chooseCar/
* License: MIT
* Used technologies:
 - AngularJS
 - Web Storage
 - FireBase
 - GruntJS
 - Bootstrap 3
